# Fonte dos dados

**Dataset:** Stack Overflow Annual Developer Survey 2024
**Fonte oficial:** https://survey.stackoverflow.co/2024/
**Espelho usado (Kaggle):** https://www.kaggle.com/datasets/berkayalan/stack-overflow-annual-developer-survey-2024
**Data de coleta original:** maio de 2024
**Tamanho:** aproximadamente 65.000 respostas, 114 colunas
**Licença:** Open Database License (ODbL) — uso livre com atribuição

## Como obter os dados

Os arquivos brutos não estão versionados neste repositório (são grandes
demais para o GitHub, e dados brutos não devem ser versionados em Git).

Para reproduzir:
1. Baixe em: https://www.kaggle.com/datasets/berkayalan/stack-overflow-annual-developer-survey-2024
2. Extraia o zip
3. Renomeie e coloque em `dados/brutos/`:
   - `pesquisa_salarial_2024.csv`
   - `pesquisa_salarial_2024_dicionario.csv`
